How to import to OneNote (easiest path)

  1. Open OneNote desktop app (Windows/Mac).
  2. Use generated file(s):
    • If you got a ZIP, unzip it first (folders mirror your ENEX notebook names).
    • If you got one DOCX, drag it directly.
  3. Drag DOCX file(s) into the target OneNote section.
  4. OneNote creates one page per document automatically.

Note: OneNote does not support direct import of native .one files from ENEX. DOCX drag-and-drop is the most reliable route while preserving structure as much as possible.
